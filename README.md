# Basic Python Programs

This repository contains my first Python programs while learning Python.

## Programs Included

### 1. Personalized Greeting
This program asks the user for their first and last name and prints a welcome greeting.

Example:
Enter your first name: sarthak
Enter your last name: shree  

Output:
Hello, sarthak shree! Welcome to Python programming.

---

### 2. Basic Mathematical Operations
This program takes two numbers as input and performs:

- Addition
- Subtraction
- Multiplication
- Division

Example:

Enter the first number: 10  
Enter the second number: 5  

Output:

Addition: 15  
Subtraction: 5  
Multiplication: 50  
Division: 2
